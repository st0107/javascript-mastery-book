# Project Catalog

Volume 6 will develop production-level projects. Each project must include architecture, folder structure, implementation, deployment, testing, and interview explanation.

## Projects

| Project | Engineering Focus |
| --- | --- |
| Task Manager | CRUD, state management, filtering, persistence |
| Chat Application | Realtime transport, optimistic UI, message ordering |
| Kanban Board | Drag and drop, accessibility, local and remote sync |
| Netflix-Style Media Browser | Data fetching, caching, responsive layout |
| GitHub Dashboard | API integration, rate limits, error handling |
| AI Chat Interface | Streaming responses, cancellation, conversation state |
| Weather App | External APIs, geolocation, resilient UI |
| File Explorer | Tree structures, lazy loading, file operations |
| Blog CMS | Auth, rich text, publishing workflow |
| Realtime Notification System | Events, queues, fan-out, delivery guarantees |

## Required Project Sections

- Problem statement.
- Product requirements.
- Architecture.
- Folder structure.
- Data model.
- Implementation.
- Testing.
- Deployment.
- Performance notes.
- Security notes.
- Interview explanation.
