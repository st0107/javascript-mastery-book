# Volume 3: Asynchronous JavaScript

Volume 3 explains asynchronous JavaScript from the host environment upward.

The reader will learn callbacks, promises, async/await, the event loop, microtasks, macrotasks, fetch, streams, cancellation, workers, and concurrency control.

## Core Outcome

The reader should be able to predict async execution order, design reliable concurrent workflows, avoid race conditions, and explain scheduling behavior under interview pressure.
