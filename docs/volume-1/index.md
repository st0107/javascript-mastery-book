# Volume 1: JavaScript Fundamentals

Volume 1 builds the execution foundation required for everything else in the book.

The emphasis is not memorizing syntax. The emphasis is understanding what JavaScript creates in memory, how code is evaluated, why confusing behavior occurs, and how to explain that behavior in an interview without hand-waving.

## Completed Chapters

- [Chapter 1: JavaScript Execution Model](chapter-01-execution-model.md)

## Volume Outcomes

By the end of this volume, the reader should be able to:

- Trace simple and moderately complex JavaScript programs step by step.
- Explain execution contexts, lexical environments, scope chains, and hoisting.
- Predict type conversion behavior.
- Use functions, objects, arrays, modules, and errors in production-quality code.
- Communicate JavaScript fundamentals clearly in interviews.
