# Volume 2: Advanced JavaScript

Volume 2 focuses on the features that separate surface-level JavaScript familiarity from professional fluency.

This volume will cover closures, `this`, prototypes, classes, descriptors, generators, functional patterns, garbage collection, V8 optimization, and design patterns. Each chapter will connect language behavior to memory, object layout, engine assumptions, and interview follow-ups.

## Core Outcome

The reader should be able to reason about object behavior, function identity, hidden state, inheritance, optimization boundaries, and maintainable abstractions in production code.
